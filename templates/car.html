Car Project

Developed an engaging Car Game Project using C# in Visual Studio, where you can use many types of cars, earn medals, and compare scores with each other and various other game features, such as realistic car physics, challenging obstacles, and dynamic game mechanics.

Tools Utilized:

  • C#: The core programming language employed in the development of the application.

  • GUI: Designed and implemented an intuitive and visually pleasing user interface to provide a seamless and enjoyable user experience.

  • Visual Studio: Utilized the development environment to create and optimize the project, ensuring smooth functionality and an engaging user experience.
