Movie Project

I spearheaded the design and development of a dynamic Movie Site, leveraging the power of Java and Spring Boot. The heart of this project lies in its sophisticated MongoDB database, meticulously engineered to efficiently store and manage an extensive library of movie data. This comprehensive platform empowers users with an array of features, including the ability to rate movies, explore captivating posters, and access trailers, all of which contribute to an enriched and immersive movie-watching experience.

Tools Utilized:

  • Java: The core programming language employed in the development of the application.

  • SpringBoot: This powerful framework streamlined the development process, allowing for rapid creation of robust and scalable applications. Its versatility and ease of use made it a key component in building a dynamic and user-friendly project.

  • IntelliJ: Utilized the development environment to create and optimize the project, ensuring smooth functionality and an engaging user experience.
